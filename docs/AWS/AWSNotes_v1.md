## Amazon S3
- AWS Storage Cloud Native options are as follows
	- 	Block
		- Amazon EBS
		- EC2 Instance Store
	- File
		-  Amazon EFS
	-  Object
		-  Amazon S3
		-  Glacier

- AWS storage gateway is a bridge between on-premise data and cloud data in S3

- Types of storage Gateways
	- File Gateway
	- Volume Gateway
	- Tape Gateway
